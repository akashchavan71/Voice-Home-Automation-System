Steps to Compile and Run the server code:

1. You will need an Arduino IDE to compile and run this code
2. Install esp8266 Board package from Boradrs Manager(under tools)
3. Install Adafruit NeoPixel library from Library Manager (under tools)
4. set the ssid & password of your wifi network in NodeMCUServerCode.ino
   Please Note: If appropriate ssid & password is not set the code might not even compile.
5. Compile the code. Connect the NodeMCU to the USB Port. Set the port of the board(under Tools) to "COM3"
6. Run the code.
